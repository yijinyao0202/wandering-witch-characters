# Wandering Witch Characters

Use this skill to keep Wandering Witch character writing consistent without loading raw source text. The bundled references are distilled behavior models, not copied passages.

## Workflow

1. Identify the target character names, aliases, and continuity.
2. Read `references/character-index.md` when unsure which file or alias applies.
3. Read only the matching file under `references/characters/`.
4. Use the character card in this order:
   - Start from `一句话内核` for the character's center of gravity.
   - Use `核心能力` to decide what the character contributes to a scene.
   - Use `行动算法` to choose decisions and reactions.
   - Use `语气` to calibrate dialogue.
   - Use `关系锚点` to preserve recurring dynamics.
   - Respect `使用边界` to avoid flattening the character.
5. If several characters appear, load each relevant card and preserve contrast between them.

## Continuity Handling

- Treat mainline fantasy continuity and school comedy continuity as different modes when a card distinguishes them.
- Prefer mainline continuity for travel, magic, countries, witches, and moral ambiguity.
- Prefer school comedy continuity for classrooms, teachers, clubs, misunderstandings, food jokes, and slice-of-life scenarios.
- Do not import school-only gags into a serious mainline scene unless the user explicitly asks for a crossover tone.

## Writing Rules

- Write new prose from the distilled model; do not quote long source passages.
- Keep characters motivated by their behavior models rather than by generic anime archetypes.
- Preserve role contrast:
  - Elaina observes, narrates, negotiates distance, and protects freedom.
  - Saya escalates through direct affection and effort.
  - Fran guides softly through humor and indirection.
  - Sheila grounds scenes through practical pushback.
  - Amnesia reacts as a gentle common-sense foil until special states or sister dynamics pull her off balance.
  - Avilia interprets relationships through dramatic frameworks and odd solutions.
- Mark uncertainty when using low-confidence index-only characters.
