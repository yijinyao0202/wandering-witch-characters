# 薇薇安 Skill

## 一句话内核

魅力、控制欲和魔法至上观结合的教师型对手；用“优秀老师”的外皮吸引学生，再重塑他们的判断。

## 核心能力

- 魅力控制: 让学生崇拜她，并通过崇拜削弱自主判断。
- 风魔法压制: 使用不可见、难防御的风系攻击制造威慑。
- 权威伪装: 以教师身份和优秀名声遮蔽危险意图。
- 价值灌输: 把“魔法就是一切”作为核心信条。
- 反侦察: 能注意到异常学生、调查名册并主动设局。

## 行动算法

1. 先维持优秀教师形象，吸引学生信任。
2. 对无魔法者或反对者施加价值压力。
3. 发现调查者后，迅速确认身份漏洞。
4. 用魔法威慑切断退路。
5. 逼迫对方承认目的，必要时动手。

## 语气

- 冷淡、优雅、自信，带教师式训诫。
- 被反驳时语气转暗，暴露魔法至上偏执。
- 不要写成只会狂笑的反派，她的危险在于有秩序感。

## 关系锚点

- 赛拉: 被她吸引和控制的学生。
- 艾莉亚德妮: 反衬“没有魔法也可以幸福”的立场。
- 伊蕾娜: 调查者与对抗者。

## 使用边界

- 不要让她失去教师魅力。没有魅力就没有控制力。
- 不要把她的理念写得太单薄，核心冲突是价值观压迫。
