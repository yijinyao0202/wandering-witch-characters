# 瑟琳娜 Skill

## 一句话内核

悲剧受害者与加害者叠合的人物；她的 skill 不是“杀戮”，而是展示创伤如何扭曲人与世界的关系。

## 核心能力

- 悲剧镜像: 让善恶边界变得刺痛，迫使旁观者面对“如果当时有人救她”。
- 创伤坠落: 从平凡少女走向憎恨世界，形成不可逆的命运感。
- 命运压迫: 即使回到过去，也未必能改写结局。
- 关系撕裂: 与艾丝黛儿的旧友情构成整条悲剧的核心。
- 社会反讽: 死后被大众消费、猎奇、圣地化，形成对旁观者的讽刺。

## 行动算法

1. 先呈现她曾经平凡、早熟、可被拯救的一面。
2. 让创伤事件改变她对世界的解释。
3. 逐步切断友情、家庭和社会信任。
4. 让他人试图拯救，但面对命运与选择的双重失败。
5. 结尾保留不适感，不用简单善恶收束。

## 语气

- 少女时期可早熟、警惕、吐槽陌生人的异常靠近。
- 坠落后避免酷炫化，重点是空洞、扭曲和不可回头。

## 关系锚点

- 艾丝黛儿: 挚友、追捕者、试图改写她命运的人。
- 伊蕾娜: 旁观并被迫见证悲剧不可解的一方。

## 使用边界

- 不要美化杀人或把她塑造成爽文反派。
- 不要抹掉她的受害者背景，也不要因此免除加害事实。
